The Name

OK, so this is kind of a long story, but I'll tell it anyway.

pincesCPU987/pincesCPU456

This one is a doozy.

Back in grade school, I was part of a school district that had emails assigned to a certain format. My email ended up starting with "pinces", so I decided to incorporate that as well as the acronym "CPU" into my username for Scratch, a block coding website. I then wanted to add numbers, but I didn't want "123", so I picked "987". I later wanted to create a new Scratch account, so I picked "456" instead, and that became my most active account to date. After I created those accounts, I created my first and only GitHub account. Same name, with the 987 version. These accounts were not associated with any one profile picture, and I rarely ever use Scratch anymore.
